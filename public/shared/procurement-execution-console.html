Workflow Guide (PR → Indent → Action Queue)

  1. PR Workbench: Generate Draft PR, edit required quantities, then Activate.
  2. Create Indent from Active PR.
  3. Indents: Approve, Issue, and Close by lifecycle status.
  4. Action Queue: Recommend vendor, Select vendor, then export buying list.
  5. Excess & Overshoot: accept excess or raise a new indent.
PR lines are locked after activation. For changes, create a new PR.
